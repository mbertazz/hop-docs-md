<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The Mapping Input transform represents and describes the input for a Mapping pipeline (referred to as a Mapping).</p>
</div>
<div class="paragraph">
<p>The data of this input will be provided by the parent pipeline (the pipeline that calls the mapping).</p>

<div class="sect1">

## Options

<div class="sectionbody">

<div class="paragraph">

The fields described in this transform are those that you want to refer to in this pipeline. When you use the mapping you can map input to these field names to arrive at a re-usable part of a pipeline.
