<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The Delay Row transform will wait the indicated timeout period before passing an input row to the next transform.</p>
</div>
<div class="paragraph">
<p>Use this transform if you deliberately want to slow down your pipeline.</p>

## Options

<div class="sectionbody">

| Option         | Description                                                                                            |
| -------------- | ------------------------------------------------------------------------------------------------------ |
| Transform name | Name of the transform. This name has to be unique in a single pipeline.                                |
| Timeout        | How long to delay. Select whether this period is measured in milliseconds, seconds, minutes, or hours. |
