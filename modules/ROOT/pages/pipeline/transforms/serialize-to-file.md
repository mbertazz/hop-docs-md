<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The Serialize To File transform stores rows of data in a binary format in a file. The binary format has an advantage over a text (flat) file because the content does not have to be parsed when read back. The metadata is stored in the file as well.</p>
</div>
<div class="paragraph">
<p>The Serialize to file transform supports a write-once access pattern, and does not support appending to existing files.</p>
</div>
