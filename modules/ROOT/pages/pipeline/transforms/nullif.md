<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The Null If transform sets the value of a certain field to null (empty) if the provided value equals a value specified in this transform.</p>
</div>
<div class="paragraph">
<p>You can add all fields from the input stream(s) using Get Fields.</p>
