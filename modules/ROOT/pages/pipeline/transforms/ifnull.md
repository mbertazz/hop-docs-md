<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The If Null transform replaces nulls by a given value either by:</p>
</div>
<div class="olist arabic">
<ol>
<li><p>Processing the complete row with all fields</p></li>
<li><p>Processing the complete row but only for specific field types (Number, String, Date etc.)</p></li>
<li><p>Processing the complete row but only for specific fields by name</p></li>
</ol>
</div>
