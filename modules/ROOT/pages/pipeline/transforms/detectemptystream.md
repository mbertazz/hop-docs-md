<h2 id="_description">Description</h2>
<div class="sectionbody">
<div class="paragraph">
<p>The Detect Empty Stream transform outputs one single empty row of data if the input stream is empty (ie when input stream does not contain any row). The output row will have the same field layout as the input row, but all field values will be empty (null).</p>
</div>
<div class="paragraph">
<p>If the input stream is not empty it will not output anything.</p>
</div>
