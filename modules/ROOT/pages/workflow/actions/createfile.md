# Create File

</div>

<div id="content">

<div class="sect1">

## Description

<div class="sectionbody">

<div class="paragraph">

The `Create File` action creates an empty file. This is useful for creating trigger files from within workflows.

</div>

</div>

</div>

<div class="sect1">
