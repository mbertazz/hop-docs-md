# Download and Install Hop

</div>

<div id="content">

<div id="preamble">

<div class="sectionbody">

<div class="paragraph">

Hop is designed to be as easy and accessible to use as possible.

</div>

<div class="paragraph">

Getting and starting Hop couldn’t be easier:

</div>

</div>

</div>

<div class="sect1">

## Download

<div class="sectionbody">

<div class="paragraph">

Download Hop from the [downloads page](http://hop.apache.org/download/).

</div>

</div>

</div>

<div class="sect1">

## Prerequisites

<div class="sectionbody">

<div class="paragraph">

Hop is self contained, all you need is a Java runtime. Hop official java version is 17. Check the [Java docs](https://java.com/en/download/help/download_options.html) to download and install a Java runtime for your operating system.

</div>

</div>

</div>

<div class="sect1">

## Install

<div class="sectionbody">

<div class="paragraph">

Unzip Hop to a folder of your choice. From that folder, you’ll be able to run the different Hop [tools](hop-tools/index.3ZCTBhjb6n) through their corresponding scripts.

</div>

<div class="paragraph">

Now that you have Hop installed, let’s have a look at all of the Hop tools you have at your disposal.

</div>

</div>

</div>
